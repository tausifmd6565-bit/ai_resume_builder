# 🎓 AI Resume Builder

An AI-powered resume builder for college students. Enter your role and skills — get a professional resume, ATS score, and skill gap analysis in seconds. **Completely free.**

## Tech Stack
- **Frontend**: HTML + TailwindCSS + Vanilla JS
- **Backend**: Python + Flask
- **AI**: Google Gemini 1.5 Flash (free)
- **Database**: Supabase (optional)

---

